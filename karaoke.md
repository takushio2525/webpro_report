# カラオケ選曲管理システム

## 概要
カラオケで歌いたい曲，得意な曲をリスト化し，選曲をスムーズにするシステム．

---

## データ構造 (Data Structure)
| フィールド | 説明 | 例 |
|-----------|------|-----|
| `id` | ID | 1, 2, 3 |
| `title` | 曲名 | 紅蓮華 |
| `artist` | アーティスト名 | LiSA |
| `genre` | ジャンル | J-POP, アニソン, 洋楽 |
| `key` | キー設定 | 原曲, +2, -1 |
| `memo` | メモ | 盛り上がる, 練習中 |
| `max_score` | 最高得点 | 94点 |
| `reserve_count` | 予約回数 | 1回 |

---

## 機能一覧

### 共通機能
- **データ管理**: 配列を用いた簡易データベース（サーバー再起動でリセットされる仕様で可）
- **一覧表示 (List)**: 登録されている全データを表形式などで表示する
- **詳細表示 (Detail)**: データの詳細情報を表示する
- **新規登録 (Create)**: 新しいデータを追加するフォームと処理
- **編集 (Edit)**: 既存のデータを修正するフォームと処理
- **削除 (Delete)**: 既存のデータを削除する処理

### 独自機能
- **検索/フィルタ**: アーティスト名やジャンルでの絞り込み
- **予約回数の編集**: 予約回数を簡単に編集できる
